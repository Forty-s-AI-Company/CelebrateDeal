# Anti-AI Design Checklist

- [ ] 畫面服務一個具體工作，不是通用 SaaS hero。
- [ ] 沒有紫藍漸層、發光球、玻璃擬態或無品牌理由的霓虹色。
- [ ] Page sections 沒有全部變成圓角卡片，卡片也沒有套卡片。
- [ ] 沒有每個標題都配彩色 icon，也沒有 emoji 充當正式 icon。
- [ ] 間距反映資訊關係，不是大量空白營造「高級」。
- [ ] 後台沒有巨大標題、巨大圓角與行銷式文案。
- [ ] 每頁只有一個主要 CTA，橘紅只用於成交/高優先行動。
- [ ] 使用真實業務情境、可辨識欄位與可行的 empty/error 指引。
- [ ] 動畫有狀態目的，且 reduced motion 下可停用。
- [ ] 行動版流程重新編排，文字、表格與按鈕不重疊或溢出。
- [ ] 繁體中文詞彙從使用者角度命名，不暴露 provider/DB 術語。
- [ ] 截圖 review 後至少移除一個無功能的視覺裝飾。
